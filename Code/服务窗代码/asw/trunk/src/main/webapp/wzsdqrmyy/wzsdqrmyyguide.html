• 挂号须知

    （一）挂号地点及挂号费：

    （1）挂号地点

    急诊病人：门诊楼1楼挂号收费处（急诊优先）

    门诊病人：门诊楼1楼挂号收费处；门诊一楼自助挂号机，支持所有门诊的挂号。

    （2）挂号费用

    普通门诊：10元

    周日普通门诊：15元

    急诊：15元

    副主任医师专家门诊：15元

    主任医师专家门诊：20元

    省级名中医专家门诊：40元

    （二）挂号时间：门诊：7:45——17：00，普通门诊11:30后挂急诊号

    急诊：24小时

    （三）挂号方法及注意事项

    （1）公费、社保病人必须带公费病历本及就诊卡挂号（限本人使用）；

    （2）初诊病人请先到服务中心填写“挂号预填单”且提供身份证或有效证件挂号；

    （3）复诊自费病人带上老病历及就诊卡挂号。

    （4）已预约的患者：有卡者请直接到自助服务机预存、挂号（首诊病人需刷身份证）；“一卡通”者请直接到自助服务机挂号，系统会自动排队；无卡者请请提前15分钟到挂号窗口挂号（报上预约时手机号码并出示身份证或有效证件、特殊照顾证件）或自助机挂号（出示身份证），如迟到系统将自动取消预约。

    （5）专家门诊实行实名挂号，必须提供身份证或有效证件挂号。如未带证件只能先挂普通门诊号就诊。

    （6）挂号未经医生刷卡，当天有效。处方24小时有效。

    （7）建议您到自助服务机或收费处预存钱，自助服务机除了现金还可以银联储蓄卡转账存钱。就诊卡内如有20元以上余额，即可直接到自助服务机刷卡挂号，系统会自动排队。

    [注]:我院均已开通电子叫号排队系统，请你在挂号时告知挂号人员挂号的科室与医生，这样你的号就直接进入电子叫号排队系统。当班医生接收挂号患者较多，医生会延时服务。第二个工作日时，电脑会将您的挂号单作废，而且您可能找不到所挂号就诊的医生。

  • 就诊

    挂号后请您到分诊台报到，在候诊区等待电子叫号排队系统呼叫到你的名字时再进入诊室就诊。如发现所挂的号与就诊科室与医师不符合时请直接到分诊台咨询。详细可咨询大厅导医。

  • 导医

    门诊楼1楼服务中心，可以为您提供：各种咨询和就诊指导；现场预约挂号；健康宣教、发放相关资料；发放化验单；代寄中药；提供针线、信封、笔、纸、老花镜、饮用水及杯子；物品保管、失物登记及招领；协助办理住院手续；出借轮椅；医疗证明盖章、登记；收集门诊患者意见等服务。您如有困难请直接与服务中心联系。

  • 收费

    全院采用微机管理，可进行各种门诊收费的电脑计价，为患者提供简便、快捷的收费方式。除了传统的窗口收费方式, 我院还提供自助机收费、诊间收费和支付宝服务窗等收费方式。“一卡通”病人可以先通过自助机或支付宝服务窗对就诊卡进行充值，在接诊过程中只要卡内金额足够就可以直接到相关科室检查、化验、取药、治疗。如需打印发票请到收费窗口打印。

  • 取药

    请携带就诊卡或发票到排队依次取药。工作时间：全年全天开放。

  • 查询

    可在门诊大厅自助机上查询我院各种药品和检查治疗费用、病人费用、充值历史、健康档案。

  • 预约

    我院门诊已全面实行预约，请您预约就诊，以减少等候时间，以免因医生的预约人数已满而挂不到当天的号，只能挂其他医生的普通号就诊。你可以通过下列方式预约：

    1、诊间预约：就诊结束后直接在医生处预约下次就诊时间（预约时间延长为45天）。

    2、现场预约：直接到门诊一楼大厅自助机预约（预约时间延长为45天）。

    3、电话预约：12580；114；医院电话（86188516）。

    4、网上预约：登陆：http://www.51gh.net（温州挂号网）注册预约。

    5、微信预约：在微信公众平台添加关注“温州市第七人民医院”。打开温州市第七人民医院认证的微信公众平台，就可以在“诊疗服务”栏目中找到“预约挂号”选择需要就诊的专家和时间进行预约。

    6、手机APP预约：温州智慧健康、支付宝服务窗

    预约注意事项：

    1、预约成功后因故不能来门诊，请及时取消。

    2、预约成功后请就诊当日提前15分钟来院取号，如迟到系统将自动取消预约。

    3、我院预约挂号采取实名制，凭身份证等有效证件挂号。
