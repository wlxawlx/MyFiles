医院介绍

温州市人民医院是一家集医疗、教学、科研、康复、预防保健于一体的三级甲等综合性医院，既有较强的综合实力、又有鲜明的专科特色。 1913年9月建成开诊，前身是天主教仁爱会创办的教会医院，1966年定名为温州市第三人民医院。1994年成为温州首家获世界卫生组织授牌的“爱婴医院”， 2000年经市政府批准筹建温州市妇产科医院，2003年经浙江省卫生厅批准与温州市妇幼保健所合署成立温州市妇幼保健院，实行温州市第三人民医院、 温州市妇幼保健院两块牌子一套班子的管理模式。2005年与温州市干部保健委员会联合组建温州市体检中心，2008年成为温州医科大学温州市第三临床学院， 2012年通过浙江省三级甲等综合性医院评审，2012年6月更名为温州市人民医院。2015年，乐清市第六人民医院及泰顺县人民医院分别挂牌成为我院“乐清分院” 及“泰顺分院”。

现医院开放床位1000余张，设28个病区，35个专业科室，11个医技科室。附设温州市体检中心、温州市职业健康检 查与职业病诊断中心、温州市医院感染管理质控中心、温州市病理检验中心、温州市产科医疗质量控制中心、温州市儿童听力诊断和治疗中心、温州市新生儿筛查中心、 温州市高危儿干预中心、温州市出血与血栓性疾病诊疗中心、温州市计划生育技术指导中心、温州市生殖健康保健中心、全市一流的母婴健康中心、儿童保健中心。 为温州市交通事故急救绿色通道中心医院。

医院专科特色鲜明，人才结构合理。血液内科为省重点学科，产科、妇科为浙南区域专病中心，妇产科和消化内科为省市共建重点学科，围产医学学科群和 消化病学科群为市级重点学科群，妇科、心血管内科、普外科、神经内科、麻醉科、骨科、急救医学、儿童保健科、口腔科、泌尿外科为市级重点学科。医院现有职工1937人， 中高级专业技术人员590人，其中高级职称 218人，不仅有一批德高望重的专家，而且涌现了一批风华正茂的后起之秀，形成了良好的学科梯队。近年来，医 院科研工作不断进步，平均每年有30余项省市课题立项，每年在国内外正式刊物上发表论文一百多篇。承担三十多所医学院校的教学任务， 并为温州医科大学国际教育学院留学生实习点。 

医院一贯坚持公立医院的公益性质，切实履行社会责任，曾先后获得省级“文明医院”、省“优质医疗、 优质服务”竞赛优胜医院、“全国百姓放心医院”等称号。与上海交通大学附属瑞金医院、上海交通大学附属胸科医院、上海复旦大学附属妇产科医院（上海红房子医院） 、浙江大学附属妇产科医院等国内著名医院建立合作关系，与美国西弗吉尼亚大学建立合作联盟。开展多形式、多渠道的学术、技术、人才交流。 2008年与新加坡国际管理学院合作开展“优质服务体系建设工程”，致力于打造温州优质服务第一品牌医院。与贵州、青海等多家医院及50多家基层医院及社区卫生 服务中心结对帮扶，切实支持边远地区和基层医疗的发展。

医院近、远期发展目标明确，前景美好。已建成的妇幼保健院综合大楼，总投资1.98亿元，建筑面积4.1万平方米，建成后将极大改善医院就诊环境， 提高医院服务能力。已开工建设的娄桥新院区占地面积130亩，建筑面积12万平方米，核定床位1000张，总投资6.8亿元。两者建成后医院将拥有2000多张床位， 真正成为集温州市人民医院、温州市妇幼保健院、温州市体检中心于一体的现代化医疗集团。

“一百年激越澎湃，一百年乘风破浪。”有着百年历史的温州市人民医院始终秉承“崇德崇新至精至诚”的院训，以“尊重、关爱、主动、沟通”的关怀准则，用心服务，用爱感动，培养高素质的医护队伍，倡导敬业爱院的团队精神，奉行廉洁行医的宗旨，营造和谐的医患关系，为打造温州优质服务第一品牌医院而努力奋斗。
